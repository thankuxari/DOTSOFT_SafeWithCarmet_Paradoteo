# DOTSOFT SafeWithCarmen Paradoteo

### Konstantinos Charitos

---

Στο project τεχνολογίες που χρησιμοποιήθηκανε React 19, σκέτο css και Material UI για μερικά μικρά components, FontAwesome για icons, ChartJS για τα charts και Vite για γρήγορο setup.

### How to run

---

Εφόσον γίνει το clone του repository σε έναν φάκελο και ο χρήστης κάνει cd σε αυτόν τα steps είναι τα εξής:

### 1. Install τα dependencies

```
    npm install
```

### 2. Run το project

```
    npm run dev
```
